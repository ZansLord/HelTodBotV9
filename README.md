# Requirements
* [Node.js](https://nodejs.org/en/)
* [Git](https://git-scm.com/downloads)
* [FFmpeg](https://www.gyan.dev/ffmpeg/builds/) (for sticker command)
* [Libwebp](https://developers.google.com/speed/webp/download) (for sticker)
